# Behavioral Guardrails

- **Logic Density**: No placeholder components. Every button in the Admin Dashboard must trigger a real DB action or state change.
- **Visual Fidelity**: Transitions must use `ease: [0.77, 0, 0.175, 1]` to match the premium Codrops feel.
- **State Management**: Use Zustand for the "Current Slide Index" to ensure the logo and counter sync perfectly with the animation state.
- **Safety**: The password `theDatum2026!` must be handled via environment variables, never hardcoded in the client-side bundle.
- **Mermaid Handling**: Always wrap Mermaid diagrams in a `Suspense` boundary to prevent layout shift during slide transitions.
